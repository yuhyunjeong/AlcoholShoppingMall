# <h2> :bulb: 프로젝트 개요</h2>
전통주 주문 및 구독형 웹사이트

다른 주류들에 비해 많이 알려지지 않은 전통주를 온라인으로 판매하여 
직접 그 지역을 방문하지 않고도 쉽게 접해볼 수 있도록 하고, 
취향에 따라 선택할 수 있도록 가이드를 제시해주며, 
사용자가 전통술에 대한 관심을 갖게 하여 추후 지역경제를 활성화시키는 것을 목표로 한다.
